# Java Fundamentals Lab

A collection of Java exercises completed as part of my Computer Science studies while expanding my software development experience beyond JavaScript and TypeScript. The repository demonstrates hands-on practice with Java syntax, object-oriented programming principles, data structures, control flow, methods, classes, and problem-solving techniques commonly used in modern software development.

## Purpose

The purpose of this repository is to document my progress learning Java and to serve as a reference for the fundamental concepts and programming patterns explored throughout my Computer Science studies.

## Topics Covered

- Java syntax
- Variables and data types
- Console output
- Conditional logic
- Loops
- Arrays
- Methods
- Classes and objects
- Basic calculations
- Program structure and organization

## How to Run

Make sure Java is installed, then compile and run any file:

```bash
javac Exercise05EvenOdd.java
java Exercise05EvenOdd
```

Replace the file name with the exercise you want to test.
